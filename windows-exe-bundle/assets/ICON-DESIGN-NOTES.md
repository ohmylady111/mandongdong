# Icon Design Notes

当前保留两种风格：

## 1) Classic（专业工具风）
- 深蓝科技感底色
- 白色漫画页叠层
- 蓝色下载箭头

## 2) Anime（当前默认）
- 紫蓝渐变底色
- 漫画页 + 分镜感
- 网点 / 速度线 / 小星闪
- 粉色下载箭头 + 青蓝高光
- 更偏二次元漫画感，但仍保持软件图标的清晰识别度

当前默认图标文件：
- `AuthorizedMangaDownloaderDesktop.ico`

保留的二次元专用文件：
- `AuthorizedMangaDownloaderDesktop-anime.ico`
- `AuthorizedMangaDownloaderDesktop-anime-256.png`
